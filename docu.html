Documentación

Área de Ayuda

Aquí te explicaremos cómo utilizar todas las funcionalidades de esta aplicación para gestionar tus tareas y asignaciones de manera eficiente.

  1. Crear una Nueva Tarea

    Paso a paso para crear una tarea: Haz clic en el botón "Crear nueva tarea". Completa los campos, como el nombre de la asignación, fecha de la asignación, tipo de prioridad y categoría. Finalmente, haz clic en "Crear tarea".

  2. Gestionar Tareas Activas

    Las tareas que has creado se muestran en una tabla con el título, fecha de entrega, prioridad, estado y categoría. Puedes eliminar, editar o marcar como completadas las tareas.

  3. Editar Tareas

    Haz clic en "Editar" para modificar una tarea existente y realiza los cambios necesarios. No olvides guardar.

  4. Eliminar una Tarea

    Para eliminar una tarea, localízala en la tabla de tareas activas y haz clic en "Eliminar". Se te pedirá confirmación.

  5. Marcar una Tarea como Completada

    Haz clic en "Completado" junto a la tarea. La tarea será marcada como terminada y se gestionará de acuerdo a su nuevo estado.

Tecnología Utilizada en la Aplicación

Backend:

  • Java con Spring Boot: Usamos el framework Spring Boot para desarrollar la lógica del backend de la aplicación. Esto nos permite gestionar las solicitudes de los usuarios, procesar los datos y comunicarnos con la base de datos de manera eficiente.
  • Base de Datos MySQL: Los datos de las tareas y asignaciones se almacenan en una base de datos MySQL, lo que garantiza una estructura sólida para manejar la información de manera segura y escalable.

Frontend:

  • HTML, CSS y JavaScript: La interfaz de la aplicación está desarrollada con HTML para la estructura, CSS para los estilos y JavaScript para la funcionalidad interactiva. Esto asegura una experiencia fluida y atractiva para los usuarios.
  • Bootstrap: Utilizamos Bootstrap como framework de diseño para lograr un estilo moderno y responsivo en toda la aplicación, adaptando la interfaz a diferentes dispositivos y resoluciones de pantalla.
  • Librería FullCalendar: El calendario que ves en la aplicación es generado utilizando la librería FullCalendar, que permite visualizar las tareas y asignaciones de una manera organizada y fácil de gestionar.

Objetivo de la Aplicación

Nuestra aplicación de cronograma está diseñada para ayudarte a tomar el control de tus tareas personales y profesionales de manera simple y eficiente. Ya sea una próxima reunión, una tarea importante o un evento personal, nuestra herramienta te permitirá gestionar cada asignación y mantenerte al día con tus compromisos.

Pero nuestro verdadero objetivo va más allá de solo organizar tu agenda. Queremos que, a través de nuestra aplicación, logres optimizar tu rendimiento diario, buscando siempre mejorar tu productividad y puntualidad. Creemos que cuando tienes el control de tus actividades, no solo eres más eficiente, sino que también cultivas el hábito del éxito. Con cada tarea completada y cada día bien organizado, estarás más cerca de convertir tus metas en realidad.
